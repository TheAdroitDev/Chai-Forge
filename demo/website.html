features

Everything you need. Nothing you don't.

⚡

Zero Build Step

No PostCSS. No Webpack. No config files. Drop one script tag and ChaiTailwind initializes itself on load.

🔭

MutationObserver

Dynamically added nodes are styled automatically. Works with SPAs, frameworks, and lazy-loaded content.

🎨

Built-in Color System

An opinionated palette with chai-bg-* and chai-text-* utilities ready out of the box.

📦

Featherlight

Pure JavaScript, zero dependencies. Optimized DOM selectors so it never scans the entire page.

🏗️

Layout Utilities

Flex, justify-content, align-items — the layout primitives you reach for every day, now as class names.

🔄

Auto-initializes

No function calls needed. ChaiTailwind listens for DOMContentLoaded and runs on its own.

Get Started

<script src="https://cdn.jsdelivr.net/gh/YOUR_USERNAME/chai-tailwind/index.js"></script>
how it works

Three sips to production.

1

Add the Script

Drop the CDN script at the bottom of your HTML. That's your entire setup.

2

Write Utility Classes

Add chai-* classes to any element. Spacing, colors, typography, flex, borders — all included.

3

Styles Applied

The engine scans, parses, converts to inline styles, and cleans up the class names. Fully automatic.
